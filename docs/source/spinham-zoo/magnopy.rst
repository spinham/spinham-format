Spin Hamiltonian
================ 

.. math::

    \mathcal{H}
    =
    \sum_{n=1}^{n=4}
    C_n
    \sum_{\substack{\mu_1,...,\mu_n,\\i_1,...,i_n}}
    J_{\mu_1,...,\mu_n}^{i_1,...,i_n}
    \prod_{m=1}^{n}
    S_{\mu_m}^{i_m}

where real constants :math:`C_n` are arbitrary and used to support any 
convention.
